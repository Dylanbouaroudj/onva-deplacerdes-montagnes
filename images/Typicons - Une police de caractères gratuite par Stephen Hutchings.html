Typicons. Icônes vectorielles parfaites et polyvalentes de 336 pixels.

Les typicons sont des icônes vectorielles gratuites à utiliser, intégrées dans une police Web et destinées à être utilisées dans vos interfaces utilisateur, que ce soit sur le Web ou dans une application native.

L'ensemble d'icônes est disponible sur Github. Tout est inclus, des fichiers source au format SVG au CSS minifed. Vous pouvez même utiliser ce projet pour créer votre propre version personnalisée de Typicons! En savoir plus ici.

Cette version améliore considérablement Typicons, avec l’ajout de 220 icônes avec des styles de contour et plein et de l’hébergement Github du contenu source. Alors qu'est-ce que tu attends? Combinez les requêtes HTTP avec une police Web conviviale compatible avec la lecture d'écran et prête pour la rétine.
